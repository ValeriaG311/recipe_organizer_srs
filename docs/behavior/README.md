# Диаграммы поведения системы "Органайзер рецептов"

## Use Case Диаграмма
Описание основных сценариев использования приложения "Органайзер рецептов"

### Акторы:
- **Пользователь** - основной пользователь системы
- **Повар** - пользователь с расширенными правами (создание рецептов)
- **Администратор** - управление системой

### Основные Use Cases:

- Управление рецептами (добавление, редактирование, удаление)
- Поиск и фильтрация рецептов
- Планирование питания
- Работа со списком покупок
- Отслеживание прогресса приготовления
- Аналитика питания
--- 
### Диаграммы последовательности

### 1. Добавление нового рецепта
Показывает процесс создания рецепта с ингредиентами и инструкциями

### 2. Отметка рецепта как приготовленного
Демонстрирует процесс завершения приготовления и обновление статистики

### 3. Фильтрация рецептов по категориям
Отображает механизм фильтрации и поиска рецептов

### 4. Ежедневный рабочий процесс
Комплексный сценарий типичного дня пользователя: от планирования до анализа

---

## Инструменты моделирования

- **PlantUML** - для текстового представления диаграмм
- **draw.io** - для визуального редактирования
